# YouTube Content Generation

Generate video concepts, scripts, and content strategies for both long-form and short-form YouTube content.

## Platform Specs

**Long-form videos:**
- Length: 8-20 minutes (sweet spot)
- Focus: Deep dives, tutorials, commentary
- Thumbnail: 1280 x 720px
- Title: 60 characters max

**Shorts:**
- Length: Under 60 seconds
- Format: 1080 x 1920px (9:16)
- Focus: Quick tips, reactions, hooks

## Content Types

### Tutorial/How-To
Educational content teaching a specific skill or process.

**Structure:**
```
Hook (0:00-0:15)
- Promise what they'll learn
- Show the end result

Context (0:15-1:00)
- Why this matters
- Who it's for

Main Content (1:00-8:00)
- Step-by-step breakdown
- Screen recordings + voiceover
- Visual examples

Recap (8:00-9:00)
- Quick summary
- Next steps

CTA (9:00-end)
- Subscribe
- Related video
- Links in description
```

**Example:**
```
Title: "Building a Design System from Scratch (No Bloat)"

Hook:
"Most design systems are over-engineered.
I'll show you how to build one that actually gets used."

[Shows polished end result]

"Let's build it."
```

### Commentary/Analysis
Opinion-based content analyzing trends, tools, or industry topics.

**Structure:**
```
Hook (0:00-0:20)
- Bold claim or question
- Why you care

Setup (0:20-1:30)
- Context
- Current situation

Analysis (1:30-7:00)
- Your perspective
- Supporting evidence
- Counter-arguments

Conclusion (7:00-8:30)
- Key takeaways
- What to do with this info
```

**Example:**
```
Title: "AI Won't Replace Designers (But This Will)"

Hook:
"Everyone's worried about AI replacing designers.
They're looking in the wrong direction."

[Visual of AI tools]

"The real threat isn't AI. It's this."
```

### Build Log
Document your process building something over time.

**Structure:**
```
Hook (0:00-0:30)
- Current state
- What you're building

Challenge (0:30-2:00)
- Problem you're solving
- Why it's hard

Process (2:00-8:00)
- What you built
- Decisions you made
- Problems you hit

Results (8:00-10:00)
- What's working
- What's not
- Next steps
```

**Example:**
```
Title: "Building an AI Creativity Tool: Week 4 Progress"

Hook:
"4 weeks building an AI tool.
No users yet.
Here's what I learned."
```

### Tool Review/Comparison
Evaluate tools and share honest opinions.

**Structure:**
```
Hook (0:00-0:30)
- What you're reviewing
- Quick verdict

Overview (0:30-2:00)
- What it does
- Who it's for

Deep Dive (2:00-8:00)
- Features breakdown
- Real-world usage
- Pros and cons

Comparison (8:00-10:00)
- vs alternatives
- Who should use it

Verdict (10:00-end)
- Final thoughts
- Worth it or not
```

## YouTube Shorts Strategy

**Types:**

**Quick Tips:**
```
Hook (0-2 sec)
Value (2-15 sec)
CTA (15-20 sec)
```

**Reactions:**
```
Show content (0-5 sec)
Your take (5-15 sec)
Conclusion (15-20 sec)
```

**Before/After:**
```
Before (0-8 sec)
Process (8-12 sec)
After (12-20 sec)
```

## Voice and Style

**Tone:**
- Conversational and direct
- Confident but not arrogant
- Educational without being condescending
- Authentic and unscripted feel

**Delivery:**
- No fluff or filler
- Get to the point
- Use examples
- Show, don't just tell

**What NOT to do:**
- Clickbait titles
- Stretched content (padding)
- Over-editing
- Fake enthusiasm
- Generic advice

## Video Production

**Equipment (minimum):**
- Screen recording (OBS, Loom)
- Decent mic (Blue Yeti, Rode)
- Basic editing (DaVinci Resolve free)

**Visual style:**
- Clean, minimal
- Dark theme preference
- Code/design focused
- B-roll when relevant

**Editing style:**
- Jump cuts for pacing
- Text overlays for key points
- Visual examples frequently
- No unnecessary transitions

## SEO Strategy

**Title formula:**
```
[Outcome] [Method/Tool] ([Qualifier])

Examples:
"Build a Design System (Without Over-Engineering)"
"AI for Designers (What Actually Works)"
"Code Your Portfolio (No Framework Needed)"
```

**Description template:**
```
[2-3 sentence summary]

Timestamps:
0:00 - Intro
0:30 - [Section]
2:00 - [Section]
...

Links:
[Relevant resources]
[Your other content]

Tools mentioned:
[List with links]
```

**Tags:**
- 3-5 primary keywords
- 3-5 related topics
- 2-3 broad categories

## Thumbnail Strategy

**Effective elements:**
- Your face (if relevant)
- Clear text (4-6 words max)
- High contrast
- Consistent style

**Colors:**
- Use your brand colors (carbon, cyan, amber)
- High contrast text
- Dark background

**Text:**
- Large, readable font
- No more than 6 words
- Key benefit or curiosity gap

## Engagement Strategy

**Comments:**
- Heart all comments in first 24 hours

- Ask follow-up questions
- Pin best comment

**Community:**
- Post updates in Community tab
- Polls for next video topics
- Behind the scenes
- Link to new videos

**Cross-promotion:**
- Mention related videos
- End screen suggestions
- Playlist organization
- Link to other platforms
