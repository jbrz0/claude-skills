# Social Media Manager Skill

Your personal social media manager for creating authentic, engaging content across multiple platforms.

## What It Does

- **Research** - Find trending topics in your interest areas
- **Generate** - Create platform-specific posts that sound like you
- **Schedule** - Build content calendars with optimal posting times
- **Visualize** - Provide Midjourney prompts and asset guidance
- **Balance** - Distribute content across your topic categories

## Quick Start

**Single post:**
```
"Give me a post for X about AI creativity"
"Create a LinkedIn update on product design trends"
```

**Multiple platforms:**
```
"Post for X and LinkedIn about productivity automation"
"Give me X thread and Instagram caption about minimalism"
```

**Content schedule:**
```
"Create X schedule for this week"
"Give me a month of LinkedIn posts"
```

**Post ideas:**
```
"3 post ideas about cyberpunk aesthetics"
"What should I post about this week?"
```

## Platforms

### X/Twitter
- Short posts (<280 chars)
- Threads (3-5 tweets)
- Conversational and punchy
- High engagement focus

### LinkedIn
- Professional updates (max 1300 chars)
- Thought leadership
- Industry insights
- Professional yet personable

### Instagram
- Visual-first
- 3 punchy bullet points
- Scannable and emotional
- Community-focused

### YouTube
- Video concepts
- Long-form and short-form ideas
- Educational and creative

## Content Types

- **Announcements** - New projects, updates
- **Curation** - Sharing resources, tools
- **Humor** - Light, relatable content
- **Insights** - Tips, observations
- **Questions** - Engagement starters
- **Reactions** - Commentary on trends
- **Resources** - Tool recommendations
- **Shoutouts** - Occasional recognition
- **Trends** - What's happening now
- **Tutorials** - How-to content

## Writing Style

✅ **Do:**
- Keep it short and simple
- Be conversational and direct
- Sound human, not AI
- Use hyphens (never en/em dashes)
- Provide value in every post

❌ **Don't:**
- Use generic phrases ("Don't miss this!")
- Sound corporate or stuffy
- Copy-paste across platforms
- Use excessive emojis
- Create engagement bait

## Best Practices

**Consistency:**
- Post regularly (schedule helps)
- Maintain voice across platforms
- Balance topic frequencies

**Engagement:**
- Reply to comments
- Start conversations
- Ask questions

**Value:**
- Every post should provide value
- Teach, entertain, or inspire
- No fluff

**Authenticity:**
- Share real experiences
- Be honest about wins and losses
- Show personality

## Getting Started

1. **Define your goals** - What do you want to achieve with social media?
2. **Start with X** - Build consistency on one platform first
3. **Research topics** - Use topic-research to find what's trending
4. **Create schedule** - Weekly calendar keeps you consistent
5. **Engage** - Reply to comments and build relationships

Ready to create content? Just ask for a post, schedule, or ideas.
